# Projeto-Calculadora
Calculadora de IMC

Participante:

Vinicius Lang


Aplicativo Android em Kotlin com exatamente duas telas. A primeira tela recebe peso e altura do usuário. Ao clicar em Calcular IMC, o app abre a segunda tela mostrando o resultado do IMC e sua classificação.

 O que foi feito
- Criação da MainActivity com campos EditText para peso e altura.
- Criação da ResultadoActivity para exibir TextViews com IMC e classificação.
- Navegação entre telas usando Intent.
- Botão Voltar usando finish().
- Layout com ConstraintLayout e IDs semânticos.


Como executar
1. Abra o Android Studio.
2. Clique em File > Open.
3. Selecione a pasta Mobile_CalculadoraIMC.
4. Aguarde sincronizar o Gradle.
5. Execute em um emulador ou celular Android.
